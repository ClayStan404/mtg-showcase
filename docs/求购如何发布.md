# 求购如何发布

- **在线文档**：[WPS求购](https://www.kdocs.cn/l/cvvaN21e3gm8)  
- 本地备份：`templates/WPS求购模板.xlsx`  
- 或手写：`wants/*.txt`

## 怎么建自己的表

1. 打开上面的云文档  
2. 在 **「模板」** 上 **右键 → 创建副本**  
3. **重命名**为自己的名字  
4. 若不能编辑 / 不能建副本：联系 QQ **417592443** 申请权限  
5. 只改自己的表  

## 统一卡行

```text
[Nx] 系列 编号 [语言] [闪] [必须此版] [| 备注]
```

| 字段 | 规则 |
|------|------|
| 语言 | `e`/`z`/`j`/`o`（空默认 e） |
| 闪 | 空/`0`=否 · `1`=是 |
| 必须此版 | 空/`0`=其他版本也可以 · `1`=必须此印刷 |
| 数量 | 空=1；或行首 `2x` |

### 例子

```text
mh3 224 e 0 1
2x msc 211 z 1 0 | 可替
iko 185 e 0 0
```
