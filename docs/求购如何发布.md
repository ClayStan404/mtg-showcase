# 求购如何发布

- 在线模板（导入 WPS）：[`templates/WPS求购模板.xlsx`](../templates/WPS求购模板.xlsx)  
- 或手写：`wants/*.txt`

## 文件头 / 表上方

昵称、城市、联系（必填）。

## 统一卡行（指定印刷 + 是否可替）

```text
[Nx] 系列 编号 [语言] [闪] [必须此版] [| 备注]
```

| 字段 | 规则 |
|------|------|
| 语言 | `e`=英 `z`=中 `j`=日 `o`=其他（空默认 e） |
| 闪 | 空/`0`=否 · `1`=是 |
| 必须此版 | 空/`0`=**其他版本也可以** · `1`=**必须此印刷** |
| 数量 | 行首 `2x` 或表「数量」列；空=1 |
| 备注 | 可选，`|` 后或表格「备注」列 |

### 例子

```text
mh3 224 e 0 1              # 只要 MH3 #224 英非闪
2x msc 211 z 1 0 | 可替    # 参考这张图/版本，其他印刷也行，要 2 张
iko 185 e 0 0              # 任意版本即可（以该印刷为参考展示）
```

## 生成网站数据

```bash
python3 scripts/build_wants.py
```

产物：`data/wants.json`、`assets/wants-data.js`。
