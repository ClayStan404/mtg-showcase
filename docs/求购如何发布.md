# 求购如何发布

数据放在仓库 `wants/*.txt`（以后可接 WPS，格式相同）。

## 文件头

```text
# buyer: 昵称
# city: 城市
# contact: 微信/QQ
```

## 两种行

### 1. 指定印刷（同卖牌）

```text
[Nx] 系列 编号 [语言] [闪]
mh3 224 e 0
2x msc 211 z 1
```

### 2. 任意版本

```text
any <牌名> [数量] [| 备注]
any Lightning Bolt | 任意版本，优先非闪
any 税赋压身 2 | 简中优先
```

## 生成网站数据

```bash
python3 scripts/build_wants.py
```

产物：`data/wants.json`、`assets/wants-data.js`。
