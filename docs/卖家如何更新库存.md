# 卖家如何更新库存（WPS）

- **在线文档**：[WPS库存](https://www.kdocs.cn/l/cgyl3WizNfp7)  
- 本地备份：`templates/WPS库存协作模板.xlsx`

## 怎么建自己的表

1. 打开上面的云文档  
2. 在 **「模板」** 工作表上 **右键 → 创建副本**  
3. **重命名**为自己的名字  
4. 若不能编辑 / 不能建副本：联系 QQ **417592443** 申请权限  
5. 只改自己的表，填昵称、城市、联系 + 卡牌行

|------|------|
| 语言 | `e`=英 `z`=中 `j`=日 `o`=其他（空默认 e） |
