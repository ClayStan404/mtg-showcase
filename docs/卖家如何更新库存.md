# 卖家如何更新库存（不用 GitHub）

面向普通用户：会用 Excel / WPS 即可。

## 你要做的

1. 下载模板：[`templates/库存填写模板.xlsx`](../templates/库存填写模板.xlsx)
2. 打开 **「我的库存」** 表，删掉灰色示例，填自己的卡
3. 保存为 `.xlsx`（推荐）
4. **私聊发给群 QQ 机器人**（或按群公告方式提交）
5. 看机器人回复：
   - 有错误 → 按行号改表，再发一次
   - 成功 → 已收下；**一般每天固定时间统一上网站**（不是立刻刷新）

## 表格列

| 列 | 必填 | 说明 |
|----|------|------|
| 卖家昵称 | 是 | 网站上显示的名字 |
| 城市 | 是 | 如武汉 |
| 联系方式 | 是 | 微信等，买家在意向清单里能看到 |
| 系列缩写 | 是 | 如 mh3、2x2 |
| 收集编号 | 是 | 如 224 |
| 语言 | 是 | 下拉：简中 / 英文 / 日文… |
| 是否闪卡 | 是 | 下拉：是 / 否 |
| 数量 | 是 | 正整数；卖光了请删行 |
| 备注 | 否 | 仅自己备忘 |

## 重要约定

- **每次提交 = 你当前全部可售库存**（以这份表为准，覆盖旧数据）
- 卖掉的卡：**删除该行**，不要填 0
- 同一印刷（系列+编号+语言+闪）合并一行，用「数量」表示多张
- **不需要** GitHub 账号，也看不到仓库

## 网站上何时能看到

由群主 / 机器人在 **每天固定时间** 汇总所有人的表，统一生成网页。  
紧急情况可找管理员「立即发布」（若开通）。
