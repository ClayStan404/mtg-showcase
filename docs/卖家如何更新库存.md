# 卖家如何更新库存（金山 WPS · 不用 GitHub）

## 管理员（你）

1. 下载模板：[`templates/WPS库存协作模板.xlsx`](../templates/WPS库存协作模板.xlsx)
2. 用 **WPS / 金山文档** 打开 → **导入到云文档**（或另存为云文档）
3. 复制工作表 **「模板_复制我」**，改名为卖家 id（如 `claystan`、`xiaoming`）
4. 填好该表顶部的：卖家昵称、城市、联系方式
5. 给对应卖家 **仅自己那张表的编辑权限**（平台若不能按表授权，则一人一个独立云文档）
6. 每天固定时间用脚本拉取云文档 → 校验 → 更新网站（卖家无感）

## 卖家

1. 打开管理员给你的表格链接（WPS 云文档）
2. **只改自己的那一张表**
3. 上方橙色格：昵称、城市、联系方式
4. 下方表格：一行一种印刷（系列 / 编号 / 语言 / 是否闪 / 数量）
5. 卖光了：**删行**；多张同样的卡：改 **数量**
6. 保存即可，一般 **每天统一更新到网站**（不是秒级）

## 表结构（每人一张 Sheet）

**卖家信息（表头区域）**

| 卖家昵称 | 城市 | 联系方式 |
|----------|------|----------|

**卡牌列表**

| 系列缩写 | 收集编号 | 语言 | 是否闪卡 | 数量 | 备注（可选） |
|----------|----------|------|----------|------|--------------|

语言、是否闪卡请用下拉选择。
