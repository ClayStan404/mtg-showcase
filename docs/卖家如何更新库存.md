# 库存表（WPS）

模板：`templates/WPS库存协作模板.xlsx`

## 列

| 系列 | 编号 | 语言 | 闪 | 数量 | 备注 |
|------|------|------|----|------|------|

表上方另填：**昵称、城市、联系**。

## 简写

| 字段 | 规则 |
|------|------|
| 语言 | 空=英文 · `z`=中文 · `j`=日文 · `o`=其他（仅这四种） |
| 闪 | 空或 `0`=非闪 · `1`=闪 |
| 数量 | 空=1 |

卖光删行。同卡一行，用数量。

管理员复制「模板」表给每人；卖家只改自己的表。每天定点同步网站。

## 开发者：从 WPS 导出更新网站

```bash
# 1) 从 WPS 导出 xlsx 后校验/写入 inventory/
python3 scripts/parse_wps_excel.py ~/下载/库存.xlsx --dry-run   # 只检查
python3 scripts/parse_wps_excel.py ~/下载/库存.xlsx            # 写入 inventory/*.txt

# 2) 生成网站数据
python3 scripts/build_data.py
```
